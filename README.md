# bivarmap

Package to create a bivariate cloropleth map.

Functions originally created by [Jos√© Hidasi Neto](https://rfunctions.blogspot.com/) and available in this [post](https://rfunctions.blogspot.com/2015/03/bivariate-maps-bivariatemap-function.html). Then implemented by in that [repository](https://gist.github.com/scbrown86/2779137a9378df7b60afd23e0c45c188) by [Prof. Stuart Brown](https://gist.github.com/scbrown86).

Install:

```{r}
install.packages("devtools")
devtools::install_github("mauriciovancine/bivarmap", ref = "HEAD")
```
